pyfian.time_value.interest_incomeÔÉÅ

interest_income.py

Interest Income Calculation ModuleÔÉÅ

This module provides functions to calculate expected interest income for a given period using different types of interest rates:

  • Continuous compounding

  • Effective annual rate

  • Nominal rate (periodic compounding)

  • Nominal rate (custom day count)

  • Money Market Rate (add-on and discount)

  • Bond Equivalent Yield (BEY)

All calculations are per dollar by default, but the principal (notional) can be customized.

FormulasÔÉÅ

  • Continuous compounding: \(I = N \times (e^{rt} - 1)\)

  • Effective annual: \(I = N \times ((1 + r)^{t} - 1)\)

  • Nominal (periodic): \(I = N \times \frac{r}{n} p\)

  • Nominal (days): \(I = N \times r \times \frac{d}{y}\)

  • Money Market Rate (discount): \(I = N \times r \times \frac{d}{b}\)

  • Money Market Rate (add-on, notional): \(I = \frac{N}{1 + r d / b} \times r \times \frac{d}{b}\)

  • Money Market Rate (add-on, investment): \(I = N \times r \times \frac{d}{b}\)

  • BEY: \(I = N \times \frac{r}{2} \times p\)

Where:

  • \(N\) = notional

  • \(r\) = rate (as decimal)

  • \(t\) = time in years

  • \(n\) = periods per year

  • \(p\) = number of periods

  • \(d\) = days in period

  • \(y\) = days in year

  • \(b\) = base days for year (e.g., 360)

Examples

>>> interest_income_continuous(0.05, 1)
0.05127109637602412
>>> interest_income_effective(0.05, 1)
0.05
>>> interest_income_nominal_periods(0.06, 12, 6)
0.03
>>> interest_income_nominal_days(0.06, 30, 360)
0.005
>>> interest_income_money_market_discount(0.06, 180)
0.03
>>> interest_income_money_market_addon_notional(0.06, 180)
0.02912621359223301
>>> interest_income_money_market_addon_investment(0.06, 180)
0.03
>>> interest_income_bey(0.06, 2)
0.06

pyfian.time_value.interest_income.interest_income_money_market_addon_notional(mmr: float, mmr_days: int, base: float = 360, notional: float = 1.0) → float[source]ÔÉÅ

Calculate interest income using a Money Market Rate (add-on) for a given period. This function assumes the notional is the total amount to be paid (face value).

The formula used is:

\[Interest = \frac{Notional}{1 + r d / base\_year} \times r \times \frac{d}{base\_year}\]

where

  • \(Notional\) is the notional amount (face value).

  • \(r\) is the Money Market Rate (as decimal).

  • \(d\) is the number of days in the period.

  • \(base\_year\) is the convention of days in the year (default 360).

Parameters:

  • mmr (float) ‚Äì Money Market Rate (add-on, as decimal).

  • mmr_days (int) ‚Äì Number of days in the period.

  • base (float, optional) ‚Äì Base days for the year (default 360).

  • notional (float, optional) ‚Äì Notional (face value) amount (default 1.0).

Returns:

Interest income for the given period.

Return type:

float

Examples

>>> interest_income_money_market_addon_notional(0.06, 180)

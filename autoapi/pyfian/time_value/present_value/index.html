pyfian.time_value.present_value.present_value_annuity(payment: float, rate: float, periods: int) → float[source]ïƒ

Calculate the present value of a fixed annuity.

The present value of a fixed annuity is given by:

\[PV = P \times \frac{1 - (1 + r)^{-n}}{r}\]
where:

  • \(PV\) is the present value

  • \(P\) is the payment per period

  • \(r\) is the interest rate per period

  • \(n\) is the total number of periods

  • payment (float) â€“ The fixed payment amount per period.

  • rate (float) â€“ The interest rate per period (as a decimal).

  • periods (int) â€“ The total number of periods.

Returns:

float

pyfian.time_value.present_value.present_value_annuity_annual(payment: float, annual_rate: float, years: int, payments_per_year: int) → float[source]ïƒ

Calculate the present value of a fixed annuity with an annual interest rate and a specified number of payments per year.

The present value is calculated as:

\[PV = P \times \frac{1 - (1 + r)^{-N}}{r}\]

where:

  • \(P\) is the payment per period

  • \(r\) is the periodic interest rate (annual_rate / payments_per_year)

  • \(N\) is the total number of periods,

with \(N = \text{years} \times \text{payments_per_year}\)

This function adjusts the interest rate and number of periods for non-annual payment frequencies.

Parameters:

  • payment (float) â€“ The fixed payment amount per period.

  • annual_rate (float) â€“ The annual interest rate (as a decimal).

  • years (int) â€“ The total number of years.

  • payments_per_year (int) â€“ The number of payments per year.

Returns:

Present value of the fixed annuity.

Return type:

float

Examples

>>> present_value_annuity_annual(100, 0.05, 10, 12)
9428.135032823473
pyfian.time_value.present_value.present_value_growing_annuity(payment: float, rate: float, periods: int, growth: float = 0.0) → float[source]ïƒ

Calculate the present value of a growing annuity.

The present value is calculated as:

\[PV = P \times \frac{1 - \left(\frac{1 + r}{1 + g}\right)^{-n}} {\left(\frac{1 + r}{1 + g}\right) - 1}\]
where:

  • \(P\) is the payment at time t=0

  • \(r\) is the interest rate per period

  • \(g\) is the growth rate per period

  • \(n\) is the total number of periods

Note

The payment parameter corresponds to the payment at time t=0. Growth is applied in the first period as well, so the payment at time t=k is:

payment * (1 + growth)^(k+1)

for each period k.

Parameters:

  • payment (float) â€“ The initial payment amount per period.

  • rate (float) â€“ The interest rate per period (as a decimal).

  • periods (int) â€“ The total number of periods.

  • growth (float, optional) â€“ The growth rate of the payments (as a decimal). Defaults to 0.0.

Returns:

Present value of the growing annuity.

Return type:

float

Examples

>>> present_value_growing_annuity(100, 0.05, 10, 0.02 )
855.5867765481578
>>> present_value_growing_annuity(100, 0.05, 10, 0.05)
1000.0
pyfian.time_value.present_value.present_value_growing_perpetuity(payment: float, rate: float, growth: float) → float[source]ïƒ

Calculate the present value of a growing perpetuity (infinite growing annuity).

The present value of a growing perpetuity is given by:

\[PV = \frac{P \times (1+g)}{r - g}\]
where:

  • \(PV\) is the present value

  • \(P\) is the payment per period

  • \(r\) is the interest rate per period

  • \(g\) is the growth rate per period

Note

The payment parameter corresponds to the payment at time t=0. Growth is applied in the first period as well, so the payment at time t=k is:

payment * (1 + growth)^(k+1)

for each period k.

Parameters:

  • payment (float) â€“ The initial payment amount per period.

  • rate (float) â€“ The interest rate per period (as a decimal).

  • growth (float) â€“ The growth rate of the payments (as a decimal).

Returns:

Present value of the growing perpetuity.

Return type:

float

pyfian.time_value.present_value.present_value_two_stage_annuity(payment: float, rate1: float, rate2: float, periods1: int, periods2: int) → float[source]ïƒ

Calculate the present value of a two-stage annuity.

The present value is calculated as:

\[PV = PV_{\text{stage1}} + PV_{\text{stage2}}\]
where:

  • \(PV_{\text{stage1}}\) is the present value of the first stage annuity

  • \(PV_{\text{stage2}}\) is the present value of the second stage annuity

The present value of the first stage is calculated using the present_value_annuity function, and the second stage is calculated using the present_value_annuity function, discounted back to the present using the interest rate of the first stage.

Note

The payment parameter corresponds to the payment at time t=0. Growth is applied in the first period as well, so the payment at time t=k is:

payment * (1 + growth)^(k+1)

for each period k.

  • payment (float) â€“ The fixed payment amount per period.

  • rate1 (float) â€“ Interest rate for the first stage (as a decimal).

  • rate2 (float) â€“ Interest rate for the second stage (as a decimal).

  • periods1 (int) â€“ Number of periods in the first stage.

Calculate the present value of a two-stage annuity where the first stage is a (possibly growing) annuity and the second stage is a (possibly growing) perpetuity.

The present value is calculated as:

\[PV = PV_{\text{stage1}} + PV_{\text{stage2}}\]
where:

  • \(PV_{\text{stage1}}\) is the present value of the first stage annuity

The present value of the first stage is calculated using the present_value_growing_annuity function, and the second stage is calculated using the present_value_growing_perpetuity function. The perpetuity is discounted back to the present using the interest rate of the first stage.

Note

The payment parameter corresponds to the payment at time t=0. Growth is applied in the first period as well, so the payment at time t=k is:

payment * (1 + growth)^(k+1)

for each period k.

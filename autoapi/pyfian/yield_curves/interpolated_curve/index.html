InterpolatedCurve represents a yield curve for zero-coupon rates at different maturities.

The curve can be set using a dictionary of zero-coupon rates and supports interpolation for maturities not explicitly provided.

It can also be derived setting a group of maturities and a group of bonds with prices, where the bonds are used to infer the zero-coupon rates using the maturities as pivots for an interpolated curve.

Parameters:

  • curve_date (str or datetime-like) ‚Äì Date of the curve.

  • maturities (list of float, optional) ‚Äì List of maturities (in years) to make curve pivot points. If zero_rates is provided, maturities will be inferred from its keys. If both maturities and zero_rates are None, default maturities will be used: [0.5, 1, 2, 3, 5, 7, 10, 20, 30].

  • zero_rates (dict, optional) ‚Äì Dictionary mapping maturities (in years) to zero-coupon rates (as decimals). If provided, maturities will be inferred from its keys.

  • bonds (list, optional) ‚Äì List of bonds used to infer the zero-coupon rates.

  • day_count_convention (str or DayCountBase, optional) ‚Äì Day count convention to use (default is None). If None, ‚Äúactual/365‚Äù will be used.

  • yield_calculation_convention (str, optional) ‚Äì Yield calculation convention to use (default is None). Supported conventions: ‚ÄúAnnual‚Äù, ‚ÄúBEY‚Äù, ‚ÄúContinuous‚Äù. If None, ‚ÄúAnnual‚Äù will be used.

Example

import pandas as pd
from pyfian.yield_curves.zero_coupon_curve import ZeroCouponCurve

zero_rates = {
    1: 0.04,   # 1 year maturity, 4% rate
    2: 0.042,  # 2 year maturity, 4.2% rate
    5: 0.045,  # 5 year maturity, 4.5% rate
}
curve_date = "2025-08-22"
curve = InterpolatedCurve(zero_rates=zero_rates, curve_date=curve_date)
# Get discount factor for 2 years
df = curve.discount_t(2)
# Get rate for 2 years
rate = curve.get_rate(2)
print(f"Discount factor for 2 years: {df}")
print(f"Zero-coupon rate for 2 years: {rate}")

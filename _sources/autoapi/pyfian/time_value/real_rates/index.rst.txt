.. py:function:: fisher_real_rate(nominal_rate: float, inflation_rate: float) -> float

   Compute the real interest rate using the approximate Fisher equation.

   Formula
   -------

   .. math::
       r_{real} \approx r_{nominal} - r_{inflation}

   :param nominal_rate: Nominal interest rate as a decimal (e.g., 0.05 for 5%).
   :type nominal_rate: float
   :param inflation_rate: Expected inflation rate as a decimal (e.g., 0.02 for 2%).
   :type inflation_rate: float

   :returns: Approximate real interest rate as a decimal.
   :rtype: float

   .. rubric:: Examples

   >>> fisher_real_rate(0.05, 0.02)
   0.03


.. py:function:: fisher_exact_real_rate(nominal_rate: float, inflation_rate: float) -> float

   Compute the real interest rate using the exact Fisher equation.

   Formula
   -------

   .. math::
       r_{real} = \frac{1 + r_{nominal}}{1 + r_{inflation}} - 1

   :param nominal_rate: Nominal interest rate as a decimal.
   :type nominal_rate: float
   :param inflation_rate: Expected inflation rate as a decimal.
   :type inflation_rate: float

   :returns: Exact real interest rate as a decimal.
   :rtype: float

   .. rubric:: Examples

   >>> round(fisher_exact_real_rate(0.05, 0.02), 6)
   0.029412
